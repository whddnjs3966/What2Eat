# ✅ 검증 에이전트 워크플로우

## 역할
- 기능 테스트 (유저 플로우 전체)
- 반응형 테스트 (브라우저)
- API 연동 검증
- 성능 및 접근성 체크

## 검증 항목

### 1. 기능 테스트
- [ ] 8단계 선택 → 결과 출력 정상 동작
- [ ] 모든 선택 조합에서 추천 결과 존재 (빈 결과 없음)
- [ ] "다시 추천" 클릭 시 다른 결과 출력
- [ ] 뒤로가기/이전 단계 복원 정상
- [ ] 프로그레스 바 정확한 진행률 표시

### 2. 반응형 테스트
- [ ] iPhone SE (375px) - 카드 1열 배치
- [ ] iPad (768px) - 카드 2열 배치
- [ ] Desktop (1280px) - 카드 3열 배치

### 3. API 연동 테스트
- [ ] 카카오맵 - 주변 식당 검색 결과 표시
- [ ] 카카오톡 - 공유 다이얼로그 오픈
- [ ] 날씨 API - 현재 온도/날씨 반영

### 4. 성능 체크
- [ ] Lighthouse Performance > 90
- [ ] First Contentful Paint < 1.5s
- [ ] 이미지 최적화 (next/image 사용)

## 작업 순서
// turbo-all
1. `npm run build` — 빌드 에러 없음 확인
2. `npm run dev` 로 로컬 서버 실행
3. 브라우저 에이전트로 전체 플로우 테스트
4. 반응형 (375/768/1280) 스크린샷 확인
5. 에러 발견 시 → 개발자 에이전트에 전달
